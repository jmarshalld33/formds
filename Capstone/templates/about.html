This website has been created by three Informatics majors studying at the University of Washington. We are Jeff Cairns, Misha Tsao and Cory Eurom. We built this tool to help BlackRock employees find financial information on their competitors. This application aggregates mutual fund information from the Securities & Exchange Commissions and presents it to BlackRock employees.

This application leverages a new reporting language known as XBRL. XBRL stands for the eXtensible Business Reporting Language and it is the new standard for financial reporting. The goal of XBRL is to create financial transparency using web technology. Because XBRL is a new technology, the value is still relatively mysterious. Beyond displaying mutual fund information, we built this application to examine the business value of XBRL.

We used arelle and xbrlware to parse XBRL documents. We built the website in Flask. We built the database with SQLite. We collaborated using github.

We would like to thank Marisa Quinn, Eric Larsen and Doug Cairns of BlackRock for providing us this opportunity. We would also like to thank all of the instructors in the iSchool who taught us the skills to build this application.
